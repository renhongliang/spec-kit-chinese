# 支持

## 如何提交问题并获取帮助

本项目使用 GitHub issues 来跟踪错误和功能请求。请在提交新问题之前搜索现有问题以避免重复。对于新问题，请将您的错误或功能请求作为新问题提交。

如需帮助或有关使用此项目的问题，请：

- 打开 [GitHub issue](https://github.com/github/spec-kit/issues/new) 报告错误、请求功能或询问有关规范驱动开发方法论的问题
- 查看[综合指南](./spec-driven.md)了解规范驱动开发流程的详细文档
- 查看 [README](./README.md) 获取开始使用说明和故障排除提示

## 项目状态

**Spec Kit** 正在积极开发中，由 GitHub 员工**和社区**维护。我们将尽力及时响应支持请求、功能请求和社区问题。
